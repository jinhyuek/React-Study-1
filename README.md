기능 구현
1. 글 목록 보기 기능(리스트 형태)
-> PostList, PostListItem
2. 글 보기 기능 
-> Post
3. 댓글 보기 기능
-> CommentList, CommentListItem
4. 글 작성 기능
->  PostWrite
5. 댓글 작성 기능
-> CommentWrite

디자인
1. 글 작성 화면(제목입력, 내용 입력, 작성 완료 버튼)
2. 메인화면(글 작성 버튼, 글 리스트)
3. 글 보기 화면(글 내용 및 댓글, 댓글 작성, 댓글 작성 완료 버튼)

        -> ui # UI Component 들을 모아놓은 폴더
